About Issue Manager

This application is designed to help users manage issues effectively. It allows users to create, view, and manage issues in a user-friendly interface.

Features

  • User authentication (login/signup)
  • Create and manage issues
  • View issue details
  • Responsive design using Bootstrap

Technologies Used

This application is built using Django, a high-level Python web framework, and utilizes Bootstrap for responsive design.
